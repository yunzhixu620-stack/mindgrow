# MindGrow UX 问题清单 & 迭代计划

## 用户反馈的问题
1. ❌ 自动补全节点太多，无法选择
2. ❌ 只有一个思维导图，不能新建平行的多个
3. ❌ 页面无法清空

## 竞品对标分析

### Xmind 做得好的：
- 左上角有明确的「新建导图」入口
- 右键菜单支持新建/删除/重命名
- 画布可以双指缩放、平移
- 节点样式丰富（颜色、图标、标记）
- 导出功能（图片、PDF、Markdown）

### mindlib 做得好的：
- 对话式创建，自然不生硬
- 多个导图并存，用图谱导航
- 自动 Wikidata 补全是可选的，不强制

### Obsidian 做得好的：
- 左侧文件列表管理多个笔记
- 搜索功能强大
- 清晰的侧边栏结构

## 迭代任务

### P0 - 必须修复（用户体验断裂）
- [ ] 自动补全改为：最多显示3个，每个可单独取消勾选
- [ ] 新增「新建导图」功能（左上角 + 按钮）
- [ ] 左侧增加导图列表侧边栏
- [ ] 新增「清空导图」功能
- [ ] 新增「删除节点」功能（选中节点后可删除）
- [ ] 确认对话框：删除/清空前二次确认

### P1 - 重要优化
- [ ] 节点右键菜单（编辑/删除/改色/子节点）
- [ ] 搜索节点功能
- [ ] 导出为图片
- [ ] 移动端适配（响应式布局）

### P2 - 锦上添花
- [ ] 节点颜色选择器
- [ ] 撤销/重做
- [ ] 节点折叠/展开
